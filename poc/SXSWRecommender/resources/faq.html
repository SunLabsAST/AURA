Paul's SXSW Artist Catalog

Frequently Asked Questions

  1. What is this? - this is a directory of artists that will be attending this year's SXSW Music Festival. It includes information the artist, including a brief artist bio, links to listen to the artist, links to see the artist's videos, links to see the artist's photos and Last.fm social tags that have been applied to the artist. You can browse the artists in alphabetical order, by popularity by tag, or via a map.
  2. What is SXSW? From the SXSW Music Site: The SXSW MUSIC AND MEDIA CONFERENCE features a legendary festival showcasing more than 1,800 musical acts of all genres from around the globe on over eighty stages in downtown Austin. By day, the Austin Convention Center comes alive with conference registrants doing business in the Trade Show and partaking of a full agenda of informative, provocative panel discussions featuring hundreds of speakers of international stature. In its 23rd year, SXSW remains an essential event on the music industry calendar.
  3. Where does the info come from? The artist list comes from the SXSW site: Offical SXSW Showcasing bands . The artist info comes from Last.fm (via their ultra-nifty web services), and the links to videos were created using the Youtube Data API. To create the google maps, the artist's hometown was geo-encoded using Google's geocoding web service, and the maps were built using the Google Maps API. Photos are shown using the simple Flickr slide show URL scheme. The 'hotness' metric is obtained via the mighty fine Echo Nest developer APIs
  4. What is this 'Hotness'? This is a somewhat experimental addition that shows how 'hot' an artist is, based on the current level of buzz around an artist in the music blogosphere. The Hotness metric comes from the musical brain at The Echo Nest. If no 'hotness' metric for an artist is shown, it doesn't necessarily mean that the band is not hot, it could just mean that we couldn't find find the data.
  5. What is this 'Rising'? This shows the artists that are 'Hot' but have few listeners. These are the bands that are getting a lot of buzz, but are still relatively unknown. (Thanks to Chris Gaines for suggesting this metric).
  6. Why did you make this? I am going to be attending SXSW this year - so I wanted to make sure that I saw all of the best bands - but I found the list of 1,200 or so artists (most of whom I had never heard of) to be rather daunting, and so I decided that I'd create a catalog that would include all the context that I'd need to decide if I'd like to see a particular band.
  7. Hey! Some of the info is wrong! Resolving artists with an API such as from Last.fm or Youtube is not an exact science. There are all sorts of reasons why things can go astray. But I'd like to fix as many errors as I can, so if you see an error, let me know.
  8. This catalog is really ugly! That's not a question. And I'm not a designer. If you have some design skills and want to make the catalog look nice, go for it.
  9. Didn't Last.fm do something like this last year? They did, and it was much better than this catalog since it gave you personalized recommendations. I hope they do it again this year too. Update Taylor McKnight has tweaked the site's CSS, so it looks much better. Thanks Taylor!
