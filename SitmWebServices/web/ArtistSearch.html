ArtistSearch

Searches the database for an artist with a particular name

Params

  • name – the name of the artist. Wildcards are allowed. Required argument
  • max – the maximum number of results returned. Optional argument, default: 10
  • outputType - the level of output detail. Optional, default is 'tiny'. Value must be one of: 'tiny', 'small', 'medium', 'large' or 'full'

Example

Request

http://music.tastekeeper.com/api/ArtistSearch?name=deer*&max=20
