AddListener

Adds a listener to the database

Params

  • appKey – the application key. Required
  • userKey – the openID of the user to be added. Required
  • lastfmName – the Last.fm name for the user. Optional
  • pandoraName – the Pandora name for the user. Optional

Example

Request

    http://music.tastekeeper.com/api/AddListener
       ?appKey=YOUR_APP_KEY
       &userKey=http://paul.lamere.myopenid.com
       &lastfmName=lamere

Results


<AddListener>
    <request>
        <param name="userKey">http://foo.lamere.myopenid.com</param>
        <param name="appKey">SAMPLE_KEY</param>
    </request>
    <results status="OK">
        <time ms="55"/>
    </results>
</AddListener>
