FindSimilarArtists

Finds artists that are similar to a seed artist.

Params

  • name – the name of the artist. Wildcards are allowed. If multiple artists match the name, the best matching artist is used. Required if key not present.
  • key – the musicbrainz ID for the artist. Required if name not present
  • max – the maximum number of results returned. Optional, default is 10.
  • popularity – the desired range of popularity in the results. Optional, default is 'all'. Value must be one of:
    • all – all artists are considered
    • head – only very popular artists are considered
    • mid – only moderately popular artists are considered
    • tail – only long tail artists are considered
    • head_mid – very and moderately popular artists are considered
    • mid_tail – very popular artists are not considered.
  • field - the field to use for similarity. Optional, default is 'socialTags'. Value must be one of:
    • all – consider all fields for similarity
    • autoTags – consider autotags for similarity
    • biographySummary – use the artist biography for similarity
    • collaborations – use the artist collaborations for similarity
    • events – use the artist
    • followers – use the artist followers for similarity
    • influencers – use the aritst influencers for similarity
    • relatedArtists – use the related artists for similarity
    • socialTags- use the social tags for similarty
    • bioTags – use bio tags (the artist biography filtered for music-related terms)
    • blurbTags – use blurb tags (artist reviews filtered for music-related terms)

For the future – we will allow specification of multiple weighed fields for similarity like so: field=(autoTags,1)(socialTags,2)(blurbTags,.5)

Example

Request

http://music.tastekeeper.com/api/FindSimilarArtists?name=weezer&popularity=head
