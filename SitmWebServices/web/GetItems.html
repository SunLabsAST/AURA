GetItems

Get items from the database

Params

  • key â€“ the key of the item of interest. Required. This may be either a single key or a list of comma separated keys.
  • outputType - the level of output detail. Optional, default is 'tiny'. Value must be one of: 'tiny', 'small', 'medium', 'large' or 'full'

Example

Request

http://music.tastekeeper.com/api/GetItems
          ?outputType=small&key=6fe07aa5-fec0-4eca-a456-f29bff451b04, \
              1e477f68-c407-4eae-ad01-518528cedc2, \

              http://static.flickr.com/33/49490624_fecd23a894.jpg, \
              50c213ed-a109-4677-9b7b-d57ef580dd2d, \
              1f8bdf5d-aed1-45aa-bf1f-94af1d7e71c1
