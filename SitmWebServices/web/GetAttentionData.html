GetAttentionData

Gets attention data from the database. This method will find all of the attention that matches the given set of parameters. If a parameter is omitted, then that parameter is ignored when looking for a match.

Params

  • max – the maximum number of attention elements to return
  • srcKey – the key of the source of the attention (typically a user/listener). Optional.
  • tgtKey – the key of the target of the attention. Optional.
  • type – the type of the attention. Optional. May be one of:
    • DISLIKED – the source doesn't like the target
    • PLAYED – the source played the target
    • LOVED – the source loves the target
    • RATING – the source rated the target
    • TAG – the source tagged the target
  • svalue – the context-dependent string value associated with the attention. Optional. Only appropriate for attention type TAG.
  • nvalue – the context-dependent integer value associated with the attention. Optional. Only appropriate for attention type RATING or PLAYED.

Example

Request

http://music.tastekeeper.com/api/GetAttentionData
    ?srcKey=http://paul.lamere.myopenid.com/&tgtKey=6fe07aa5-fec0-4eca-a456-f29bff451b04
    &max=5
