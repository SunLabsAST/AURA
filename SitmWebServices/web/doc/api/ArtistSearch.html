ArtistSearch

Searches the database for an artist with a particular name

Params

  • name – the name of the artist. Wildcards are allowed. Required argument
  • max – the maximum number of results returned. Optional argument, default: 10

Example

	http://music.tastekeeper.com/api/1.0/ArtistSearch?name=beat&max=20
