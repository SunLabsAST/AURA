AddArtist

Adds an artist to the database. Note that when the call returns the artist will be added to the database, but it may take some time before all data sources have been crawled, indexed and ready for use.

Params

  • appKey – the application key. Required
  • mbaid – the musicbrainz ID of the artist to be added. Required

Example

Request

    http://music.tastekeeper.com/api/AddArtist
        ?appKey=SAMPLE_APP_ID
        &mbaid=688d084b-2b22-44bb-ae69-1a41499eb82f

Results


<AddArtist>
