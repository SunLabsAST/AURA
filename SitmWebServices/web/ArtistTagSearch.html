ArtistTagSearch

Searches the database for an artist tag with a particular name

Params

  • name – the name of the artist tag. Wildcards are allowed. Required argument
  • max – the maximum number of results returned. Optional argument, default: 10

Example

Request

        http://music.tastekeeper.com/api/ArtistTagSearch?name=fem*&max=10
