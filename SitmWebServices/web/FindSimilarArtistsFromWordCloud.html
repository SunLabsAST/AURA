FindSimilarArtistsFromWordCloud

Finds artists that are similar to a word cloud

Params

  • wordCloud – the word cloud. Required. The word cloud is of the form:
    • Simple:
      tag+name+1,tag+name2, ...
    • Or with weights
      (tag name 1, weight1)(tag name 2, weight 2) ...
  • max – the maximum number of results returned. Optional, default is 10.
  • popularity – the desired range of popularity in the results. Optional, default is 'all'. Value must be one of:
    • all – all artists are considered
    • head – only very popular artists are considered
    • mid – only moderately popular artists are considered
    • tail – only long tail artists are considered
    • head_mid – very and moderately popular artists are considered
    • mid_tail – very popular artists are not considered.
  • outputType - the level of output detail. Optional, default is 'tiny'. Value must be one of: 'tiny', 'small', 'medium', 'large' or 'full'

Example

Request

http://music.tastekeeper.com/api/FindSimilarArtistsFromWordCloud
    ?wordCloud=(indie,1)(punk,1)(emo,.5)&popularity=head

http://music.tastekeeper.com/api/1.0/FindSimilarArtistsFromWordCloud
    ?wordCloud=jazz,blues,cool,mello,saxophone
