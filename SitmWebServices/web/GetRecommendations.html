GetRecommendations

Gets recommendations for a user

Params

  • userKey – the key for the user of interest. Required.
  • max – the maximum number of recommendations returned. Optional, default is 10.
  • alg – the name of the algorithm to use. Optional, default is defined in GetRecommendationTypes
  • popularity – the desired range of popularity in the results. Optional, default is 'all'. Value must be one of:
    • all – all artists are considered
    • head – only very popular artists are considered
    • mid – only moderately popular artists are considered
    • tail – only long tail artists are considered
    • head_mid – very and moderately popular artists are considered
    • mid_tail – very popular artists are not considered.

Example

Request


http://music.tastekeeper.com/api/GetRecommendations
    ?userKey=http://paul.lamere.myopenid.com/
    &alg=SimpleArtist&max=5
