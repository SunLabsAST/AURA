FindSimilarArtistTags

Finds artist tags that are similar to a seed tag.

Params

  • name – the name of the artist tag. Wildcards are allowed. If multiple tags match the name, the best matching tag is used. Required if key not present.
  • key – the ID for the tag. Required if name not present
  • max – the maximum number of results returned. Optional, default is 10.
  • outputType - the level of output detail. Optional, default is 'tiny'. Value must be one of: 'tiny', 'small', 'medium', 'large' or 'full'

Example

Request

http://music.tastekeeper.com/api/FindSimilarArtistTags?name=heavy+metal

Results
