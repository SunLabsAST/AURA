GetApml

Gets the APML markup for a listener. APML is the Attention Profile Markup Language. Read more at apml.org.

Params

  • userKey – the key for the user of interest. Required.
  • max – the maximum number of concepts returned. Optional, default is 100.
  • format – the format of the artist concept section, either 'artist' or 'mbaid'. Optional, default is mbaid.

Example

http://music.tastekeeper.com/SitmWebServices/GetApml?userID=http://paul.lamere.myopenid.com&format=artist&max=5
