FindSimilarListeners

Finds listeners that are similar to a seed listener.

Params

  • key – the key for the listener. Required.
  • max – the maximum number of results returned. Optional, default is 10.
  • field - the field to use for similarity. Optional, default is 'socialTags'. Value must be one of:
    • all – consider all fields for similarity
    • artists – consider artists that the user has a preference for (via plays, ratings, or loved attention) for similarity
    • socialTags- use the social tags of the listener for similarty

For the future – we will allow specification of multiple weighed fields for similarity like so: field=(socialTags,1)(artists,2)

Example

Request

http://music.tastekeeper.com/api/FindSimilarListners
    ?key=http://lamere.myopenid.com
    &max=5

Results
