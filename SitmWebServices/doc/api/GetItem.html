GetItem

Gets an item from the database

Params

  • key – the key of the item of interest. Required. This may be either a single key or a list of comma separated keys.
  • format – the format of the output. May be 'compact' or 'full'. Optional, default is 'full'. iF 'compact' is selected, certain items (such as the artist, are output in a compact form).

Example

http://www.tastekeeper.com/api/1.0/GetItem
    ?key=http://static.flickr.com/2401/2326748577_c9f8868370.jpg

Results

The format of the results depend upon the type of item being retreived
